Configuração de HMAC
Configuração de HMAC

Configure a chave HMAC para segurança do webhook. Esta chave será usada para verificar a autenticidade do webhook.

Aviso de Segurança Importante

A chave HMAC será ocultada após o salvamento por motivos de segurança.
• Você não poderá visualizá-la novamente
• Salve uma cópia em um local seguro antes de prosseguir
• Se perdida, você precisará gerar uma nova chave

Mínimo 32 caracteres para segurança. Mantenha esta chave secreta!
